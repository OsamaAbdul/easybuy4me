# EasyBuy4Me

Empowering consumers with the freedom to shop, pay, and grow. Join the future of global retail.

EasyBuy4Me is a modern landing page for a service that handles daily errands, grocery shopping, fast food delivery, and global procurement. It features a premium, high-tech fintech aesthetic with smooth animations and interactive components.

## 🚀 Features

- **Grocery Errands**: Send your list, and we handle the market runs.
- **Fast Food Delivery**: Pickup from your favorite restaurants.
- **General Errands**: Laundry pickup, document drop-off, bill payments.
- **Instant Payments**: Seamless payment via Opay or Bank Transfer.
- **Global Procurement**: Procure and ship international items to your doorstep.
- **Trusted Service**: Verified agents and secure transactions.
- **Interactive Persona Section**: Tailored experiences for different user types with a skewed grid layout.
- **Security Focused**: Details on data encryption and payment security.
- **Smooth Scrolling**: Powered by Lenis for a premium, high-end feel.
- **Dynamic Modals**: Legal and company information accessible via interactive modals in the footer.
- **Back to Top Button**: Glowing and floating button for easy navigation.

## 🛠️ Tech Stack

- **Framework**: React (with TypeScript)
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Hugeicons (`@hugeicons/react`, `@hugeicons/core-free-icons`)
- **Smooth Scroll**: Lenis

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd EASYBUY4ME
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/components/`: Contains all React components (Hero, Features, Navbar, etc.)
- `src/components/ui/`: Contains UI components like the `FooterTapedDesign`.
- `src/App.tsx`: The main application component assembling all sections.
- `src/main.tsx`: Entry point for the application.
- `tailwind.config.cjs`: Tailwind CSS configuration with custom design tokens (colors, fonts, etc.).

## 🌐 Deployment

This project is configured for deployment on **Vercel**. A `vercel.json` file is included in the root to handle SPA routing (rewrites to `index.html`).
